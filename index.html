Task 2

  • Що таке HTML?

    • HTML означає HyperText Markup Language.
    • Це стандартна мова розмітки для створення веб-сторінок та веб-додатків.

  • Що таке CSS?

    • CSS означає Cascading Style Sheets.
    • Це мова стилів, яка використовується для оформлення веб-сторінок.

  • Що таке JavaScript?

    • JavaScript - це мова програмування, яка використовується для створення інтерактивних веб-сторінок та веб-додатків.
    • Вона може використовуватись для створення динамічного контенту, обробки подій та взаємодії з користувачем.
